# Object-Detection-by-Drone-
Developed a real-time object detection system using Python, OpenCV, and deep learning models. The system detects and classifies objects from images or live video streams and highlights them with bounding boxes. It demonstrates practical applications of computer vision in surveillance, automation, and smart monitoring systems
Features

Real-time object detection using webcam

Image-based object detection

Bounding box visualization around detected objects

Labeling detected objects with confidence score

Technologies Used

Python

OpenCV

TensorFlow / YOLO

NumPy

Applications

Smart surveillance systems

Traffic monitoring

Autonomous vehicles

Security systems.
